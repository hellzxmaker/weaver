# Project Weaver

Prereqs:
- meteor installed (see Meteor's official website https://www.meteor.com/install)

Installation:
- Mac
 1. Clone the repo
 2. Run 'meteor npm install' on weaver/
 3. Run 'meteor' on weaver/

View:
- To view the application, go to localhost
